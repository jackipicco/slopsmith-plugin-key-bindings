# Slopsmith Key Bindings

This plugin was made to create keyboard / TV remote key bindings that trigger events within Slopsmith.

The key bindings have been default to work with a standard TV remote but can be altered within the settings menu.

The motivation for this plugin was to load Slopsmith from a Smart TV's browser.

## Default Bindings

| Action | Key Binding | Description |
| :--- | :---: | :--- |
| **Seek Back** | `1` | Rewinds the media playback by 5 seconds. |
| **Toggle Play** | `2` | Plays or pauses the media playback. |
| **Seek Forward** | `3` | Fast-forwards the media playback by 5 seconds. |
| **Set Loop Start** | `4` | Marks the beginning timestamp for a playback loop. |
| **Clear Loop** | `5` | Removes the active playback loop markers. |
| **Set Loop End** | `6` | Marks the ending timestamp for a playback loop. |
| **ARR Select** | `7` | Toggles the arrangement (e.g. Lead, Rythem) |
| **Viz Picker** | `8` | Toggles the visualisation |
| **Quality Select** | `9` | Toggles the quality. |
